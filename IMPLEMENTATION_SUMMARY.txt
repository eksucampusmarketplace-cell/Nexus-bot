BOT COMMANDS ADDED:
- /play - Play music (reply to audio/voice or send file)
- /skip - Skip current track
- /queue - View music queue
- /stop - Stop music and clear queue
- /pause - Pause playback
- /resume - Resume playback
- /nowplaying - Show current track with controls

API ENDPOINTS ADDED:
- GET /api/music/{chat_id}/queue - Get current queue
- POST /api/music/{chat_id}/command - Send command
- POST /api/music/{chat_id}/add - Add track to queue
- POST /api/music/{chat_id}/clear - Clear queue
