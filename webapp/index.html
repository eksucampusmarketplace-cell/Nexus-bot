❓ Help & Tips

Basic Commands:

  • /play - Play music (reply to audio)
  • /skip - Skip current track
  • /queue - View queue

Advanced Commands:

  • /play_youtube - Play from YouTube
  • /volume - Set volume (0-200)
  • /repeat - Set repeat mode
  • /shuffle - Toggle shuffle
  • /playlist_create - Create playlist
  • /playlist_play - Play playlist
  • /search - Search music
